订阅条款

重要提示：本订阅条款适用于「日语轻学」应用的所有自动续订订阅服务。请仔细阅读以下条款，特别是涉及费用、续订和取消的部分。

1. 订阅服务概述

1.1 服务内容

「日语轻学」提供以下自动续订订阅服务：

  • 周会员：按周订阅，享受无限制学习功能
  • 月度会员：按月订阅，享受无限制学习功能
  • 年度会员：按年订阅，享受无限制学习功能及优惠价格
  • 终身会员：一次性购买，永久享受无限制学习功能

1.2 核心功能

订阅用户可享受以下高级功能：

  • 无限制单词搜索
  • 无限制收藏夹创建
  • 精选词单无限制查看
  • 数据云端同步
  • 无广告学习环境
  • 优先客服支持

2. 价格与计费

2.1 订阅价格

当前订阅价格如下（价格可能因地区而异）：

  • 周会员：¥8/周
  • 月度会员：¥15/月（新用户首次订阅可免费获得3天）
  • 年度会员：¥148/年（新用户首次订阅可免费获得1周）
  • 终身会员：¥198（原价¥398，节省¥200）

2.2 付款方式

所有订阅费用将通过您的Apple ID账户收取，并受Apple iTunes Store条款约束。

2.3 价格变更

我们保留随时调整订阅价格的权利。如有重大价格变更，我们将提前30天通知您。

3. 自动续订

3.1 续订机制

除终身会员外，所有订阅服务均为自动续订订阅：

  • 订阅将在当前订阅期结束前24小时内自动续费
  • 续订费用将从您的iTunes账户中扣除
  • 续订周期与初始订阅周期相同

3.2 续订确认

您可以通过以下方式确认是否启用了自动续订：

  • 在设备上查看订阅设置
  • 通过iTunes Store账户设置查看

4. 取消与退款

4.1 取消自动续订

您可以随时通过以下步骤取消自动续订：

  1. 打开设备上的"设置"
  2. 点击您的Apple ID
  3. 选择"媒体与购买项目"
  4. 点击"查看账户"
  5. 选择"订阅"
  6. 找到"日语轻学"订阅
  7. 点击"取消订阅"

重要：取消订阅后，您仍可使用订阅服务直至当前订阅期结束，但不会自动续订。

4.2 退款政策

根据Apple iTunes Store政策，所有购买均为最终交易，不提供退款。如有特殊情况，请通过以下联系方式与我们联系。

5. 优惠与试用期

5.1 免费试用

新用户可能享有免费试用期，试用期时长如下：

  • 月度会员：3天免费试用
  • 年度会员：7天免费试用

5.2 试用期收费

如果您未在试用期结束前取消订阅，将按正常价格收取订阅费用。

6. 账户管理

6.1 订阅管理

您可以通过以下方式管理您的订阅：

  • iOS设备：设置 → [您的姓名] → 媒体与购买项目 → 查看账户 → 订阅
  • 网页版：https://apps.apple.com/account/subscriptions

6.2 恢复购买

如果您更换设备或重新安装应用，可以使用"恢复购买"功能重新获得订阅权益。
